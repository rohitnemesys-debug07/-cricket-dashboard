# 🏏 Cricket Analytics Dashboard

🔗 Live App: https://cricket-dashboard-rohit.streamlit.app/

---

## 📌 Project Overview
This project is a real-time cricket analytics dashboard built using Streamlit. It fetches live match data from an API, stores matches in a database, and performs SQL-based analytics.

---

## 🚀 Features
- Live cricket match updates
- Save match data to database
- SQL-based analytics
- Graph visualization
- Interactive dashboard

---

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- SQLite
- REST API
