# 🏏 Cricket Analytics Dashboard


## Project Overview
This project is a real-time cricket analytics dashboard built using Streamlit. It fetches live match data from an API, stores matches in a database, and performs SQL-based analytics.


## Tech Stack
- Python
- Streamlit
- Pandas
- SQLite
- REST API
- pandas

## Features
- Live cricket match updates using API
- player statistics and performance tracking
- SQL-based analytics (25+ queries)
- Full CRUD operations (Add, Update, Delete players)
- Interactive dashboard with filters and charts
- Cloud deployment using Streamlit

## Modules
- Home Dashboard (KPIs)
- Live Matches
- Saved Matches
- Top Players
- SQL Analytics
- CRUD Operations
- Graphs & Visualization
